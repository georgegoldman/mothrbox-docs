The Mothrbox Rust SDK provides **native, zero-dependency** integration for building fast, secure applications with encrypted decentralized storage.

Designed for performance-critical use cases, the Rust SDK compiles to WebAssembly (Wasm) for browser use or native binaries for servers/desktops.

All encryption is performed using Rust's battle-tested crypto libraries â€” no JavaScript or external runtimes required.

## Basic Usage

### Encrypt & Upload (AES-256-GCM - Default)

```rust
use mothrbox_sdk::{MothrboxClient, EncryptionMode};
use std::fs;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = MothrboxClient::new(); // Uses default gateway

    let file_data = fs::read("confidential.pdf")?;
    let password = "MyStrongRustPassword2026!";

    let upload_result = client.encrypt_and_upload(
        &file_data,
        "confidential.pdf",
        password,
        EncryptionMode::Aes256Gcm, // default
    ).await?;

    println!("Uploaded! Blob ID: {}", upload_result.blob_id);
    // Securely store blob_id for later

    Ok(())
}
```

### Download & Decrypt

```rust
use mothrbox_sdk::{MothrboxClient};
use std::fs;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = MothrboxClient::new();

    let blob_id = "abc123xyz...";
    let password = "MyStrongRustPassword2026!";

    let decrypted = client.download_and_decrypt(
        blob_id,
        password,
    ).await?;

    fs::write("recovered.pdf", decrypted.data)?;
    println!("Decrypted {} bytes -> recovered.pdf", decrypted.data.len());

    Ok(())
}
```

## Supported Encryption Modes

```rust
EncryptionMode::Aes256Gcm         // Default - maximum speed on modern CPUs
EncryptionMode::ChaCha20Poly1305   // Best software performance
EncryptionMode::EccP256           // Public-key encryption (coming soon)
```

## Key Features
- **No heap allocations** during encryption where possible
- **Zero-copy** file handling for large data
- **Wasm ready** â€” run in browser with `wasm32` target
- **Async-first** with Tokio support
- **No external crypto deps** â€” uses rustcrypto ecosystem

## Use Cases
