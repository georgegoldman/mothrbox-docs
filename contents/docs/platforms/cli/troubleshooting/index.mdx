## Upload Failed / "Upload to Walrus failed"

- Check your `.env` file has correct `SUI_SECRET_KEY`
- Verify wallet has SUI tokens:
  ```bash
  sui client balance
  ```
- Get testnet tokens: https://faucet.testnet.sui.io/
- Restart system:
  ```bash
  ./mothrbox restart
  ```

## Decryption Failed / "Decryption failed: invalid MAC"

Most common causes:
1. Wrong password (case-sensitive!)
2. Wrong encryption mode (used AES but trying ChaCha decrypt, or vice versa)
3. Wrong Blob ID (copy-paste error)

Double-check:
- Exact Blob ID
- Exact password
- Matching encrypt/decrypt command

Test with known good file if possible.
