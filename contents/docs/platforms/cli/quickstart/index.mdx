# âš¡ CLI Quickstart

Get Mothrbox running locally and encrypt your first file in under 5 minutes using the command-line interface.

## Prerequisites
- Linux, macOS, or Windows (WSL2 recommended)
- Docker & Docker Compose installed
- A Sui wallet with testnet/mainnet tokens

## Step-by-Step Setup

### 1. Clone the Repository
```bash
git clone https://github.com/georgegoldman/mothrbox_v2
cd mothrbox_v2
```

### 2. Make CLI Executable
```bash
chmod +x mothrbox
```

### 3. Configure Your Sui Wallet
```bash
cd mothrbox_ts
cp .env.example .env
nano .env  # Edit and add your SUI_SECRET_KEY
cd ..
```

### 4. Build and Start Mothrbox
```bash
./mothrbox build
./mothrbox start
```
