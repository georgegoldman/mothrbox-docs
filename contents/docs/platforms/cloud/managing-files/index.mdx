# üìÇ Managing Files

Your dashboard shows all uploaded files in a clean table view.

![File list view](/img/cloud/file-list.png)

### Features
| Action       | How To                                      | Notes                                      |
|--------------|---------------------------------------------|--------------------------------------------|
| **View**     | Click file name                             | Shows metadata, Blob ID, upload date       |
| **Download** | Click download icon                         | Decrypts automatically with your password  |
| **Delete**   | Click trash icon ‚Üí Confirm                  | Permanent ‚Äì no recovery!                   |
| **Rename**   | Click pencil icon                           | Only changes display name (not Blob ID)    |
| **Search**   | Use top search bar                          | By name or Blob ID                         |
| **Filter**   | By date, size, or encryption type           |                                            |

### Downloading a File
1. Find the file in the list.
2. Click the download icon.
3. Enter your password (or private key for ECC).
4. File saves decrypted to your downloads folder.

   ![Download prompt](/img/cloud/download-prompt.png)
