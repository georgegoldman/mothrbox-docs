# ❓ FAQs

## Can I use Mothrbox in frontend apps?
No. Never expose API keys client-side. Use Mothrbox from your backend.
## What happens if I lose the private key?
Encrypted files will be unreadable. Mothrbox cannot recover your key.
## What’s the max file size?
Currently 100MB per file. Larger limits coming soon.

## Where is the data stored?
Data is stored across edge and decentralized storage networks, optimized for performance.
