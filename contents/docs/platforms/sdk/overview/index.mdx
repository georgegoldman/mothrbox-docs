The Mothrbox SDKs allow developers to integrate military-grade encrypted decentralized storage directly into their applications.

All SDKs provide the same core features as the CLI and Cloud dashboard:
- End-to-end encryption in the client
- Support for AES-256-GCM, ChaCha20-Poly1305, and ECC P-256
- Direct upload/retrieval from the Walrus decentralized network
- Simple Blob ID management
